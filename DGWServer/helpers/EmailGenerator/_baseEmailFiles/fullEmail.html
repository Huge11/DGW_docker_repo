Learn The Fretboard!

Most guitarists never learn the names of the guitar frets.  Although it isn't necessary if you're just learning pop songs, learning all the notes on the fretboard is one of the first steps to really mastering the guitar. 

Hint: You can remember the names of the open strings with this phrase: 

Eddie Ate Dynamite, Good Bye Eddie

Find...

The G note - on the 6th string

The D note - on 2nd string

What is the Name of...

The 10th fret - on the 3rd string

The 2nd fret - on the 4th string
